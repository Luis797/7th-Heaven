Requirements

 You will need the following to run 7th Heaven w/ Final Fantasy 7:
Legally Purchased Final Fantasy 7 Game: (1998 Original, 2012 Square Enix Store, or 2013 Steam release)
Operating System: Windows Vista SP2 or newer. Windows 8 and newer highly recommended.
Software Dependency: Microsoft .NET Framework 4.5.2 (your PC may already have this installed or is built into your OS)
Processor (CPU): 2 GHz or greater
Memory (RAM): 1 GB or greater
Graphics Card (GPU): Dedicated graphics supporting OpenGL 2.0. 512+ MB of video RAM preferred.
*Integrated graphics may work, but with slow performance, especially with high resolution textures/models.

Installing Final Fantasy 7

Steam Version (Recommended)


1. Click 'Next'.



2. Select 'English'. Click 'Next'.



3. Install to 'C:\Steam'. Click 'Next'.
DO NOT INSTALL UNDER A 'PROGRAM FILES' FOLDER! This can cause mods to break!
If you've installed Steam and/or Final Fantasy VII under a system folder like 'Program Files', then 7H will prompt you to copy the game to 'C:\Games\Final Fantasy VII'.



4. 'Run Steam' should be checked. Click 'Finish'.



5. Type in your Steam credentials and click 'Login'. Follow any instructions given by Steam.





6. Find 'Final Fantasy VII' in your games Library list. Select it and click 'Install'.



7. Click 'Next'.



8. Click 'I Agree'.



9. Click 'Finish'. You're done!


1998 Disc Version


1. Insert your FF7 Install disc. Navigate to the drive in Windows/File Explorer, then double-click 'setup.exe'.
If you receive a 'Do you want to allow this app...' prompt, select 'Yes'.



2. Click 'Install' at the top. You do NOT need to install any of the other options.



3. Click 'Next'.



4. Uncheck everything. Windows already includes DirectX and music is now in OGG format. Click 'Next'.



5. Select 'Maximum Install'. Click 'Next'.
If you accidentally selected 'Standard Install', 7H can automatically fix this for you later.



6. Click 'Browse'.
DO NOT INSTALL UNDER THE DEFAULT FOLDER OR A 'PROGRAM FILES' FOLDER! This can cause mods to break!
If you've installed Final Fantasy VII under a system folder like 'Program Files', then 7H will prompt you to copy the game to 'C:\Games\Final Fantasy VII'.



6b. Here, it is easiest to type in 'C:\Games\Final Fantasy VII' into the box (without quotes). Click 'OK'.



6c. Click 'Yes'.



6d. Your destination folder should look like the screenshot. Click 'Next'.



7. Click 'Next'.



8. Enjoy the music while your game installs.



9. Click 'OK'. Ignore the message instructions. You're done!
It is recommended you copy the FF7 movies to your computer using the Import Movies From Disc feature.


Installing 7th Heaven

1. 7th Heaven installer splash screen.



2. Click 'Next'.



3. Select 'I accept the agreement' then click 'Next'.



4. Click 'Browse' if you wish to select a different folder to install 7H to. Otherwise, click 'Next'.



5. Uncheck one or both checkboxes if you don't want to create shortcuts. Click 'Next'.



6. Optionally open 7th Heaven after install by leaving the 'Launch 7th Heaven' box checked. Click 'Finish'.
You can always later open 7th Heaven from your Start Menu or Desktop (if you created shortcuts).
Otherwise, run it from the installed folder\7th Heaven.exe.


7. Within 7th Heaven, download, install, and activate your desired mods. Change your settings in the Settings menu if you like, and click the 'Play' button! That's it!
Check out the Using 7th Heaven section to learn how to use 7th Heaven's features.
